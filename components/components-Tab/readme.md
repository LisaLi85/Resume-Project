Tab选项卡组件

    参数说明
        container: Tab，
        triggerType: 'click','mouseover'
        activeStyle: active时的样式名称
        chooseTabHead: 选择TabHead的字符串
        chooseTabContent: 选择TabContent的字符串 

    兼容性
        IE8及以上
        Firefox 3.5及以上
        Chrome4及以上
        Safari 3.1及以上
        Opera10.1 及以上
